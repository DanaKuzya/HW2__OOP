Название: Рецепты и списки покупок!!!
Краткое описание: 
Консольная программа для управления проектами
Данная программа позволяет создавать рецепт с порционным количеством ингредиентов, добавлять ингредиенты.
Поддерживаюются так же диетические рецепты.
Можно создавать списки покупок, объединяя разные рецепты на разное количество персон и списки покупок.

Использование:
Клонирование репозитория:
git clone https://github.com/DanaKuzya/HW2__OOP
cd HW2__OOP
Установка зависимостей:
pip install -r requirements.txt
Запуск тестов:
pytest
